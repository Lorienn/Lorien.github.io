Node.js

（一）入门

  1. 含义：Node.js是跨平台的js运行环境，基于V8引擎

  2. 和浏览器的区别

    浏览器

    • 和DOM、Web平台的API（cookie）进行交互
    • 需要使用Babel将代码转换为ES5
    • 遵循ES模块规范（import）

    Node

    • 没有浏览器提供的相关API
    • 支持ES6-9
    • 遵循CommonJS规范（require）

  3. 基本操作

  4. 模块系统

  5. npm

    • 安装（npm install pkg）

      --save(-S)：添加到package.json的dependencies，生产环境依赖

      --save-dev(-D)：添加到package.json的devDependencies，仅用于开发，在生产环境中不需要，例如测试软件包、webpack、babel

      -g：安装至全局位置。使用npm root -g命令查看全局位置
